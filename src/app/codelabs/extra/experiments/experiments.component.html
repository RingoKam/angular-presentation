How to use angular-presentation

    first you create the slide with

    <slides-slide>
      ...
    </slides-slide>

    this component creates new empty separate slide for presentation

HTML tags in slide component

Inside this component you can use regular html tags like h1, h2, p, ol, li and others.

<h2> Text of header </h2>

slides-code-editor

    This component lets you to display code in the slide.

    <slides-code-editor [fontSize]="30" [code]="code.moduleAnatomy.code"
           [focus-highlight-match]="[code.moduleAnatomy.matches.ngModule]">
    </slides-code-editor>
